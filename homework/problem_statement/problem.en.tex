\problemname{wholelottahomework}

Your little sibling is in 2nd grade and needs help with their math homework. 
For a few days, it goes well; however, due to bureaucratic restrictions, the teacher cannot give them more challenging problems.
As a result, the teacher assigns increasingly large numbers of the same type of problems.

When faced with up towards 100,000 problems each day, helping your sibling has become quite the hassle.
Being the smart programmer you are, you decide to create a program that solves these problems automatically, and frees you from this commitment.

\begin{Input}
One line with integer n (1 \le n \le 100_000), the number of problems.
$n$ lines, each with two integers $a$ and $b$, a mathematical operator $op$ in the format of 'a op b' where: 
0 <= a <= 99
0 <= b <= 99
op is either + or -


\end{Input}
\begin{Output}
\end{Output}
